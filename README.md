# 📌 **Project Overview**

Encryption_Decryption is a web-based cybersecurity project that demonstrates three classical encryption algorithms used to securely transform readable text into encrypted text and back again.

The application allows users to encrypt and decrypt messages using **Caesar Shift**, **Vigenère**, and **Atbash** ciphers. It also includes an interactive cipher wheel, live substitution mapping, and automatic history tracking to help users understand how classical cryptography works.

# 🚀 **Features**

✅ Encrypt plaintext into ciphertext

✅ Decrypt ciphertext back into plaintext

✅ Interactive cipher visualization

### **Supported Encryption Methods**

- Caesar Shift Cipher
- Vigenère Cipher
- Atbash Cipher

### **Application Features**

- Interactive Caesar cipher wheel
- Live substitution key mapping
- Preserve letter case option
- Copy encrypted text
- Automatic encryption/decryption history
- History stored using Local Storage
- Fully responsive interface
- Works completely in the browser
- No server or database required
